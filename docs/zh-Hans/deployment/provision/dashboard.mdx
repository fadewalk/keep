---
title: "仪表板配置"
---

在 Keep 中配置仪表板允许您配置和管理数据可视化。本节将指导您完成设置和配置仪表板所需的步骤。

### 仪表板配置概述

Keep 中的仪表板使用定义布局、数据源和可视化组件的 JSON 字符串进行配置。这些配置可以通过环境变量或配置文件进行管理。

### 环境变量

要配置仪表板，您需要设置以下环境变量：

| 环境变量 | 用途 |
| -------------------- | ----------------------------------------------- |
| `KEEP_DASHBOARDS` | 包含仪表板配置的 JSON 字符串 |

### 配置示例

```json
          "w": 3,
          "h": 3,
          "minW": 2,
          "minH": 2,
          "static": false
        }
      ],
      "widget_data": [
        {
          "i": "w-1728223503577",
          "x": 0,
          "y": 0,
          "w": 3,
          "h": 3,

          "minH": 2,

          "thresholds": [

```

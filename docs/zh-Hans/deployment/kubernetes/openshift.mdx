Keep 的 Helm Chart 也支持 Openshift 安装。

只需按照 Kubernetes 设置指南操作，但请确保按如下方式修改 values.yaml 文件中 frontend(/backend).route 下的以下行：
```
enabled: true
host: <desired-hostname>
path: <desired-path> # should be / for default
tls: <desired-tls-configs>
wildcardPolicy: <desired-wildcardPolicy>
```

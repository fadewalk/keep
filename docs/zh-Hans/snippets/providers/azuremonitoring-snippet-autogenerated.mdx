## 通过 Webhook 连接（双向）
此提供商支持 webhook。


要将警报从 Azure Monitor 发送到 Keep，请使用以下 webhook URL 配置 Azure Monitor 将警报发送到 Keep：

1. 在 Azure Monitor 中，创建一个新的操作组。
2. 在操作组中，添加一个类型为"Webhook"的新操作。
3. 在 Webhook 操作中，使用以下设置配置 webhook。
- **名称**: keep-azuremonitoring-webhook-integration
- **URL**: 您的 Keep 后端 URL
4. 保存操作组。
5. 在警报规则中，配置操作组使用步骤 1 中创建的操作组。
6. 保存警报规则。
7. 测试警报规则以确保警报正在发送到 Keep。

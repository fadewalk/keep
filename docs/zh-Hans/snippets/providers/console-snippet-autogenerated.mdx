## 在工作流中

此提供商可用于工作流。


作为"步骤"来查询数据，示例：
```yaml
steps:
    - name: Query console
      provider: console
      config: "{{ provider.my_provider_name }}"
      with:
        message: {value}
        logger: {value}
        severity: {value}
```


作为"操作"来进行更改或更新数据，示例：
```yaml
actions:
    - name: Query console
      provider: console
      config: "{{ provider.my_provider_name }}"
      with:
        message: {value}  # 要打印到控制台的消息
        logger: {value}  # 是否使用记录器
        severity: {value}  # 如果 logger 为 True，则为消息的严重程度
```

## 认证
此提供商需要身份验证。
- **kubeconfig**: Kubeconfig 文件内容（必需：False，敏感：True）
- **context**: 要使用的 Kubernetes 上下文（必需：False，敏感：False）
- **namespace**: Flux CD 安装的命名空间（必需：False，敏感：False）
- **api_server**: Kubernetes API 服务器 URL（必需：False，敏感：False）
- **token**: Kubernetes API 令牌（必需：False，敏感：True）
- **insecure**: 跳过 TLS 验证（必需：False，敏感：False）

某些范围可能需要才能通过提供商执行特定操作或查询。以下是相关范围及其用例的摘要：
- **authenticated**: 用户已授权（强制）

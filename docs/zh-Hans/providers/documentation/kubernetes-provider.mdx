## 连接 Provider

要连接到 Kubernetes，请按照以下步骤操作：

1. 在 Kubernetes 上创建服务账户。
2. 使用 rolebinding/clusterrolebinding 创建 role/clusterrole 并绑定到服务账户。
3. 获取服务账户的 token。

## 注意事项

- 此 provider 允许您与 Kubernetes 交互以执行滚动重启或 pod 列表操作。

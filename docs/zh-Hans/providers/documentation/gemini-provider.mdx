## 连接 Provider

要连接到 Gemini，您需要获取一个 API 密钥：

1. 进入 [Google AI Studio](https://makersuite.google.com/app/apikey)。
2. 点击 **Create API Key** 或使用现有的密钥。
3. 为 Keep 复制您的 API 密钥。

在 Gemini Provider 配置的 `authentication` 部分使用生成的 API 密钥。

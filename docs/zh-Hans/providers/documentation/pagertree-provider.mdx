## 连接 Provider

- 要与 Pagertree API 交互，您需要提供 api_token。
- 您可以在您的 [User Settings](https://app.pagertree.com/user/settings) 页面查看和管理您的 API 密钥。


## 注意事项

_此 provider 使用 Pagertree API 发送告警或根据提供的参数将其标记为事件。如果事件标记为 true，它会调用 `__send_alert` 或 `__send_incident` 方法。_

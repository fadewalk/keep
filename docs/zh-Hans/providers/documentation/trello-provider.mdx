## 连接 Provider

1. 前往 https://trello.com/power-ups/admin 创建自定义 power-up。
2. 创建新的 power-up 并添加基本详细信息，如名称、电子邮件地址等。
3. 创建后，导航至 power-up 内部并进入 API Key 部分。
4. 在那里点击 `Generate a new API key`，它将生成将用作 `api_key` 的 API Key。
5. 对于生成 `api_token`，有手动生成令牌的选项，点击该选项并授权应用程序。

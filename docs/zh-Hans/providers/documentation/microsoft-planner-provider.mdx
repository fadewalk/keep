## 连接 Provider

要连接到 Microsoft Planner，请按照以下步骤操作：

1. 登录您的 [Azure](https://azure.microsoft.com/) 账户。
2. 在此处注册应用程序 [here](https://portal.azure.com/#view/Microsoft_AAD_RegisteredApps/CreateApplicationBlade/isMSAApp~/false)。
3. 成功注册应用程序后，转到 **API permissions** 页面并添加以下权限：
    - `Tasks.Read.All`
    - `Tasks.ReadWrite.All`
4. 转到 **Overview** 页面，记下 `Application (client) ID` 和 `Directory (tenant) ID`。
5. 转到 **Certificates & secrets** 页面，创建一个新的客户端密钥并记下客户端密钥值。
6. 将客户端 ID、客户端密钥和租户 ID 添加到 Microsoft Planner Provider 配置的 `authentication` 部分。

## 注意事项

- 此 provider 允许您与 Microsoft Planner Provider 交互以创建任务。

## 连接 Provider

### API 密钥

要获取 Elastic API 密钥，请按照以下步骤操作：

1. 登录您的 elastic.co 账户。
2. 进入 "Elasticsearch Service" 部分。
3. 点击 "API Key" 按钮。
4. 生成一个新的 API 密钥。

### Cloud ID

要获取 Elastic Cloud ID，请按照以下步骤操作：

1. 登录您的 elastic.co 账户。
2. 进入 "Elasticsearch Service" 部分。
3. 在概述页面中找到 "Cloud ID"。

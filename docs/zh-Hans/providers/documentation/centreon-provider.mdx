## 连接 Provider

1. Centreon 可以是 SaaS 或本地部署。您需要有一个正在运行的 Centreon 实例。
2. 进入 Administration > API Tokens，为管理员用户创建一个新令牌。
3. 使用您的 Centreon 实例的 URL 和 API 令牌配置 provider。

## 相关链接

## 注意事项

- Centreon 仅支持以下 [主机状态](https://docs.centreon.com/docs/api/rest-api-v1/#realtime-information)（UP = 0, DOWN = 2, UNREA = 3）。

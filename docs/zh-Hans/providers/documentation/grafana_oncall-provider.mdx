## 连接 Provider

要连接到 Grafana OnCall，您需要创建一个 API 令牌：

1. 登录您的 Grafana 账户。
2. 进入 "Alerts & IRM" -> OnCall。
3. 进入 **Settings** 页面。
4. 点击 **Create** 按钮并为您的令牌提供一个名称。
5. 复制令牌值并妥善保管。
6. 在 Grafana Oncall Provider 配置的 `authentication` 部分添加令牌值。

## 注意事项

- 此 provider 允许您与 Grafana OnCall 交互以创建告警。
- Keep 将在 Grafana OnCall 内部创建一个名为 "Keep Integration" 的 "Webhook" 类型集成。

Payload 示例：

```json
{
    "alert_uid": "08d6891a-835c-e661-39fa-96b6a9e26552",
    "title": "The whole system is down",
    "image_url": "https://upload.wikimedia.org/wikipedia/commons/e/ee/Grumpy_Cat_by_Gage_Skidmore.jpg",
    "state": "alerting",
    "link_to_upstream_details": "https://en.wikipedia.org/wiki/Downtime",
    "message": "Smth happened. Oh no!"
}
```

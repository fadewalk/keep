## 连接 Provider

为了连接到 MongoDB 数据库，您可以使用连接 URI 或单独的参数。以下是提供认证信息的方法：

1. 如果使用连接 URI，请提供包含 MongoDB 连接字符串的 `host` 参数。
2. 如果使用单独的参数，请提供以下信息：
   - `username`：MongoDB 用户名。
   - `password`：MongoDB 密码。
   - `host`：MongoDB 主机名。
   - `database`：MongoDB 数据库名称。
   - `authSource`：MongoDB 数据库名称。

## 注意事项

- 确保提供的用户具有在指定 MongoDB 数据库上执行查询的必要权限。

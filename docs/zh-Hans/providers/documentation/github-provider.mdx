## 连接 Provider

1. 进入您的 GitHub 账户并导航到 **Settings > Developer Settings > Personal Access Tokens**。
2. 生成具有所需权限的令牌（例如 `repo`、`workflow` 等）。
3. 复制令牌并将其作为 `github_token` 提供在 provider 配置中。

## 概述

Monday Provider 可以与 Monday.com 无缝集成，Monday.com 是一个工作操作系统，为团队提供运行项目和工作流的信心。使用 Monday Provider，您可以向看板添加新的 pulse。

<AutoGeneratedSnippet />

#### 管理员选项卡

如果您是 monday.com 账户的管理员用户，请按照以下步骤访问您的 API 令牌：

1. 登录您的 monday.com 账户。
2. 点击右上角的个人资料/头像。
3. 选择 Administration > Connections > API。
4. 复制您的个人令牌。请注意，您可以随时重新生成新令牌，但这样做会导致之前的令牌过期。

#### 开发者选项卡

如果您是成员用户或 monday.com 账户的管理员，请按照以下步骤访问您的 API 令牌：

1. 登录您的 monday.com 账户。
2. 点击右上角的个人资料图片。
3. 选择 Developers。这将在另一个选项卡中打开 Developer Center。
4. 点击 My Access Tokens > Show。
5. 复制您的个人令牌。请注意，您可以随时重新生成新令牌，但这样做会导致之前的令牌过期。

## 将 Monday 连接到 Keep

1. 从 Monday 获取 API 令牌。
2. 在 Keep 中添加 Monday 作为 provider。
3. 为 provider 提供一个名称，将 API 令牌粘贴到 `Personal API Token` 字段中，然后点击 `Connect`。

## 使用方法

1. 为了向看板添加新的 pulse，您需要以下信息：
   - Board ID：要添加 pulse 的看板的 ID。
   - Group ID：要添加 pulse 的群组的 ID。
   - Item Name：要添加的 pulse 的名称。
   - Column Values：要为 pulse 设置的列的值。
2. 在 monday.com 应用中打开要添加 pulse 的看板。
3. 将鼠标悬停在侧面板中的看板名称上，点击出现的三个点，然后点击 ID 以复制看板 ID。
4. 将鼠标悬停在看板中的群组名称上，点击出现的三个点，然后点击 Group ID 以复制群组 ID。
5. Item Name 是您要添加的 pulse 的名称。
6. Column ID 和 Column Value 是您要为 pulse 设置的列的值。将鼠标悬停在看板中的列名称上，点击出现的三个点，然后点击 Column ID 以复制列 ID。列值是您要为该列设置的值。

## 连接 Provider

获取 UptimeKuma 主机 URL、用户名和密码

1. UptimeKuma 只能自托管。您需要运行一个 UptimeKuma 实例。
2. 设置 UptimeKuma 后，您可以获取主机 URL、用户名和密码。
3. 在 provider 设置中使用获取的主机 URL、用户名和密码。

## Webhooks 集成

1. 使用所需参数连接到 UptimeKuma provider。
2. 使用 Keep Backend API URL 作为 UptimeKuma 中的主机 URL。[https://api.keephq.dev](https://api.keephq.dev)（默认）
3. 在 Keep 中导航至账户设置，进入 API Keys，并为 Webhook 生成一个 API 密钥。

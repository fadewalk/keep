## 连接 Provider

要使用 Opsgenie Provider，您需要从 API Integration 提供 API 密钥和集成名称。您可以在 Settings -> Integrations -> Add integration 下创建 API 集成，搜索 API Integration。选择 API，为集成提供一个名称，然后点击 continue。

您可以在 Settings -> Integrations -> Add integration 下创建集成密钥

<Note>
  如果您使用的是免费版，集成密钥可以在 Teams ->
  您的团队 -> Integrations -> Add Integration (API) 下创建。
</Note>

访问 [Opsgenie API Integration](https://app.opsgenie.com/settings/integrations/create/api) 以快速创建 API 集成。

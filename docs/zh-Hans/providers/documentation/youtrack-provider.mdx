## 概述

YouTrack 是一个项目管理工具，充满了简化您工作并提高任何团队项目工作效率的功能。从软件开发和 DevOps 到 HR 和营销，各种团队都可以使用 YouTrack 的功能轻松跟踪和协作处理任何规模的项目。

### 如何获取项目 ID 和永久令牌？

1. **项目 ID**：项目 ID 可以在项目的 URL 中找到。例如，在 URL `https://<your-youtrack-host>/projects/<project-id>` 中，项目 ID 是 `<project-id>`。

2. **永久令牌**：查看 [YouTrack - 生成永久令牌](https://www.jetbrains.com/help/youtrack/server/manage-permanent-token.html) 文档以生成永久令牌。

## 连接 Provider

### 获取 API 令牌

要使用 LinearB provider，您必须从 LinearB 获取 API 令牌：

1. 登录您的 LinearB 账户。
2. 导航至 API 设置部分。
3. 使用适当的权限生成新的 API 令牌。
4. 安全存储 API 令牌，因为需要它来在 Keep 中配置 LinearB provider。

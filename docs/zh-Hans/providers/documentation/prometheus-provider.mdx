## 连接 Provider

1. 设置一个 Prometheus 服务器并确保它正在运行。
2. 获取可访问您的 Prometheus 实例的 `prometheus_url`。
3. （可选）如果受保护，从 Prometheus 配置中获取 API 令牌。
4. 在 provider 配置中提供这些值。

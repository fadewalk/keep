## 连接 Provider

1. 进入 [Personal Access Token](https://docs.gitlab.com/ee/user/profile/personal_access_tokens.html#create-a-personal-access-token) 查看如何创建 personal_access_token。
2. 获取 `host`，例如：如果您使用 Cloud GitLab，请使用：`https://gitlab.com`；如果您使用本地部署，请使用您自己的 `host`。

## 连接 Provider

1. 打开 Google Chat。
2. 打开您要添加 webhook 的聊天室。
3. 在聊天室标题旁边，点击展开更多箭头，然后点击"Apps & Integrations"。
4. 点击"+ Add webhooks"。
5. 在 Name 字段中，输入"Quickstart Webhook"。
6. 在 Avatar URL 字段中，输入 https://developers.google.com/chat/images/chat-product-icon.png。
7. 点击 Save。
8. 要复制 webhook URL，点击"More"，然后点击"Copy link"。

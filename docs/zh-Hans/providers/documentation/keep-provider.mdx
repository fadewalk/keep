## 认证参数

要使用 Keep provider，您必须使用与 Keep 账户关联的 API token 进行认证。该 token 可以从 Keep 仪表板生成。

## 连接 Provider

1. 登录您的 Keep 账户。
2. 导航至账户仪表板的 API 部分并生成 API token。
3. 通过 Keep provider 查询告警时使用此 token 进行认证。

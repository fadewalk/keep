## 连接 Provider

要连接到 DeepSeek，您需要获取一个 API 密钥：

1. 在 [DeepSeek](https://platform.deepseek.com) 注册账户。
2. 导航到您的账户设置。
3. 为 Keep 生成一个 API 密钥。

在 DeepSeek Provider 配置的 `authentication` 部分使用生成的 API 密钥。

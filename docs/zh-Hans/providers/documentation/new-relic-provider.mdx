## 连接 Provider

1. 前往 https://one.newrelic.com/admin-portal/api-keys/home 创建 User Key。
2. 从创建的密钥中获取 `api_key` 和 `account_id`。
3. 根据区域从此处获取 `api_url`：https://docs.newrelic.com/docs/apis/rest-api-v2/get-started/introduction-new-relic-rest-api-v2 。

## Webhook 集成修改

webhook 集成将 Keep 添加为 New Relic 内"Alerts and AI" API 中的目标。
这授予 Keep 对 New Relic 内以下范围的访问权限：
- `ai.destinations:read`
- `ai.destinations:write`
- `ai.channels:read`
- `ai.channels:write`

## 连接 Provider

要连接到 Axiom，您需要从您的 Axiom 账户创建一个 API 令牌。请按照以下步骤操作：

1. 登录您的 Axiom 账户。
2. 进入**设置**菜单下的 **API Access** 页面。
3. 点击 **Create Token** 按钮并为令牌输入一个名称。
4. 复制令牌值并妥善保管。
5. 在 Axiom Provider 配置的 `authentication` 部分添加令牌值。

要访问数据集，您需要提供组织 ID。您可以在 Axiom Web 应用的 URL 中找到您的组织 ID。例如，如果您的 Axiom URL 是 `https://app.axiom.co/organizations/1234`，那么您的组织 ID 是 `1234`。

## 注意事项

- 此 Provider 支持 Axiom API 提供的有限功能集。
- `startTime` 和 `endTime` 参数使用 ISO-8601 格式。
- `query` 函数以 JSON 格式从 Axiom API 返回响应。

## 概述

**ArgoCD Provider** 便于从 ArgoCD 拉取拓扑和应用程序数据。
ArgoCD 应用程序映射到 Keep Services。
ArgoCD ApplicationSets 映射到 Keep Applications。

<AutoGeneratedSnippet />

## 连接 Provider

1. 按照 [ArgoCD 用户管理文档](https://argo-cd.readthedocs.io/en/stable/operator-manual/user-management/#manage-users) 中的 `生成 auth token` 从您的 ArgoCD 实例获取**访问令牌**。
2. 将**部署 URL** 设置为您的 ArgoCD 实例的基础 URL（例如 `https://localhost:8080`）。

## 功能

**ArgoCD Provider** 支持以下主要功能：

- **拓扑**：使用 ArgoCD 中的应用程序配置拓扑。
- **应用程序**：使用 ArgoCD 中的 ApplicationSets 创建应用程序。

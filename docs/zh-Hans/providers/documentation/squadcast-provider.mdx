## 输入

`notify` 函数将以下参数作为输入：

- `notify_type`（必填）：取 `incident` 或 `notes` 之一，取决于您是要创建事件还是备注。
1. ##### `incident` 的参数
   - `message`（必填）：这将是事件消息。
   - `description`（必填）：这将是事件描述。
   - `tags`（可选）：事件的标签。它应该是字典格式。
   - `priority`（可选）：事件的优先级。
   - `status`（可选）：事件的状态。
   - `event_id`（可选）：event_id 用于解决事件
   - `additional_json`（可选）：要与事件一起发送的附加 JSON 数据。
2. ##### `notes` 的参数
   - `message`（必填）：备注的消息。
   - `incident_id`（必填）：要创建备注的事件的 ID。
   - `attachments`（可选）：备注的附件列表。

查看 [文档](https://support.squadcast.com/integrations/incident-webhook-incident-webhook-api) 了解更多

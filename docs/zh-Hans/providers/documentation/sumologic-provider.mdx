## 连接 Provider

1. 按照此处的说明获取您的 Access Key 和 Access ID：https://help.sumologic.com/docs/manage/security/access-keys/
2. 确保用户具有以下能力的角色：
    - `manageScheduledViews`
    - `manageConnections`
    - `manageUsersAndRoles`
3. 从此处找到您的 `deployment`：https://api.sumologic.com/docs/#section/Getting-Started/API-Endpoints，keep 将自动确定您的端点。

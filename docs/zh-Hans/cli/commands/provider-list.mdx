列出提供者。

## 用法

```
Usage: keep provider list [OPTIONS]
```

## 选项
* `available`:
  * 类型：BOOL
  * 默认值：`false`
  * 用法：`--available
-a`

  列出可以安装的提供者。


* `help`:
  * 类型：BOOL
  * 默认值：`false`
  * 用法：`--help`

  显示此消息并退出.



## CLI 帮助

```
Usage: keep provider list [OPTIONS]

  List providers.

Options:
  -a, --available  List provider that you can install.
  --help           Show this message and exit.
```

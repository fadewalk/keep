启动 API。

## 用法

```
Usage: keep api [OPTIONS]
```

## 选项
* `multi_tenant`:
  * 类型：BOOL
  * 默认值：`false`
  * 用法：`--multi-tenant`

  启用多租户模式


* `help`:
  * 类型：BOOL
  * 默认值：`false`
  * 用法：`--help`

  显示此消息并退出.



## CLI 帮助

```
Usage: keep api [OPTIONS]

  Start the API.

Options:
  --multi-tenant  Enable multi-tenant mode
  --help          Show this message and exit.
```

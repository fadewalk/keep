
# cli alert

管理告警。

## 用法

```
Usage: cli alert [OPTIONS] COMMAND [ARGS]...
```

## 选项
* `help`:
  * 类型：BOOL
  * 默认值：`false`
  * 用法：`--help`

  显示此消息并退出。



## CLI 帮助

```
Usage: cli alert [OPTIONS] COMMAND [ARGS]...

  Manage alerts.

Options:
  --help  Show this message and exit.

Commands:
  enrich  Enrich an alert.
  get
  list    List alerts.
```

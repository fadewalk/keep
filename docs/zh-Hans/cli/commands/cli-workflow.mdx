管理工作流。

## 用法

```
Usage: cli workflow [OPTIONS] COMMAND [ARGS]...
```

## 选项
* `help`:
  * 类型：BOOL
  * 默认值：`false`
  * 用法：`--help`

  显示此消息并退出.



## CLI 帮助

```
Usage: cli workflow [OPTIONS] COMMAND [ARGS]...

  Manage workflows.

Options:
  --help  Show this message and exit.

Commands:
  apply  Apply a workflow.
  list   List workflows.
  run    Run a workflow with a specified ID and fingerprint.
  runs   Manage workflows executions.
```

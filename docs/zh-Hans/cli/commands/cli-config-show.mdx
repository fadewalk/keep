---
sidebarTitle: "keep config show"
---

显示 Keep 配置。

## 用法

```
Usage: keep config show [OPTIONS]...
```

## 选项
* `help`:
  * 类型：BOOL
  * 默认值：`false`
  * 用法：`--help`

  显示此消息并退出.



## CLI 帮助

```
Usage: keep config show [OPTIONS]

  show the current config.

Options:
  --help  Show this message and exit.
```

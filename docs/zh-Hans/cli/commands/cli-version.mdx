获取库版本。

## 用法

```
Usage: keep version [OPTIONS]
```

## 选项
* `help`:
  * 类型：BOOL
  * 默认值：`false`
  * 用法：`--help`

  显示此消息并退出.



## CLI 帮助

```
Usage: keep version [OPTIONS]

  Get the library version.

Options:
  --help  Show this message and exit.
```

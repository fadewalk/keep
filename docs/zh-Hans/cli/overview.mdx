title: "概述"
---

Keep CLI 允许您从命令行管理 Keep。

首先请[安装](/cli/installation) Keep CLI，然后[运行工作流](/cli/commands/cli-run)。

### 环境变量

| 环境变量 | 用途 | 必填 | 默认值 | 有效选项 |
|:-------------------:|:-------:|:----------:|:-------------:|:-------------:|
| **KEEP_CLI_IGNORE_SSL** | 连接 KEEP API 时忽略 SSL | 否 | false | "true" 或 "false" |

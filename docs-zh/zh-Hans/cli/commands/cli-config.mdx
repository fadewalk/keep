设置 Keep 配置。

## 用法

```
Usage: keep config [OPTIONS] COMMAND [ARGS]...
```

## 选项
* `help`:
  * 类型：BOOL
  * 默认值：`false`
  * 用法：`--help`

  显示此消息并退出.



## CLI 帮助

```
Usage: keep config [OPTIONS] COMMAND [ARGS]...

  Manage the config.

Options:
  --help  Show this message and exit.

Commands:
  new   create new config.
  show  show the current config.
```

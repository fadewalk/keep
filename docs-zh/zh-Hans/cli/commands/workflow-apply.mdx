---

应用工作流。

## 用法

```
Usage: keep workflow apply [OPTIONS]
```

## 选项
* `file` (必填):
  * 类型：Path
  * 默认值：`none`
  * 用法：`--file
-f`

  工作流文件


* `help`:
  * 类型：BOOL
  * 默认值：`false`
  * 用法：`--help`

  显示此消息并退出.



## CLI 帮助

```
Usage: keep workflow apply [OPTIONS]

  Apply a workflow.

Options:
  -f, --file PATH  The workflow file  [required]
  --help           Show this message and exit.
```

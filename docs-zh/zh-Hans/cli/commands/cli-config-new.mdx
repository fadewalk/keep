创建新配置。

## 用法

```
Usage: keep config new [OPTIONS]...
```

## 选项
* `interactive`:
  * 类型：BOOL
  * 默认值：`True`
  * 用法：`--interactive`

  交互式创建配置。

* `url`:
  * 类型：STRING
  * 默认值：`http://localhost:8080`
  * 用法：`--url`

  Keep 后端服务器的 URL。

* `api-key`:
  * 类型：STRING
  * 默认值：``
  * 用法：`--api-key`

  用于 Keep 身份验证的 API 密钥。

* `help`:
  * 类型：BOOL
  * 默认值：`false`
  * 用法：`--help`

  显示此消息并退出.



## CLI 帮助

```
Usage: keep config new [OPTIONS]

  create new config.

Options:
  -u, --url TEXT      The url of the keep api
  -a, --api-key TEXT  The api key for keep
  -i, --interactive   Interactive mode creating keep config (default True)
  --help              Show this message and exit.
```

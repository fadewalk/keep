
# cli provider

管理提供者。

## 用法

```
Usage: cli provider [OPTIONS] COMMAND [ARGS]...
```

## 选项
* `help`:
  * 类型：BOOL
  * 默认值：`false`
  * 用法：`--help`

  显示此消息并退出.



## CLI 帮助

```
Usage: cli provider [OPTIONS] COMMAND [ARGS]...

  Manage providers.

Options:
  --help  Show this message and exit.

Commands:
  connect
  delete
  list     List providers.
```

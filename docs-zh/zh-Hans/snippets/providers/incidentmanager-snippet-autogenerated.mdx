## 认证
此提供商需要身份验证。
- **region**: AWS 区域（必需：True，敏感：False）
- **response_plan_arn**: AWS 响应计划的 arn（必需：True，敏感：False）
- **sns_topic_arn**: 您想在响应计划中使用/正在使用的 AWS SNS 主题 arn（必需：True，敏感：False）
- **access_key**: AWS 访问密钥（如果在 EC2 上使用 IAM 角色则留空）（必需：False，敏感：True）
- **access_key_secret**: AWS 访问密钥密钥（如果在 EC2 上使用 IAM 角色则留空）（必需：False，敏感：True）

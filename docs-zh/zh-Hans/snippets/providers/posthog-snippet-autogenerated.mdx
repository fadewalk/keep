## 认证
此提供商需要身份验证。
- **api_key**: PostHog API 密钥（必需：True，敏感：True）
- **project_id**: PostHog 项目 ID（必需：True，敏感：False）

某些范围可能需要才能通过提供商执行特定操作或查询。以下是相关范围及其用例的摘要：
- **session_recording:read**: 读取 PostHog 会话录制（强制）
- **session_recording_playlist:read**: 读取 PostHog 会话录制播放列表
- **project:read**: 读取 PostHog 项目数据（强制）

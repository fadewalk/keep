## 认证
此提供商需要身份验证。
- **api_key**: Datadog API 密钥（必需：True，敏感：True）
- **app_key**: Datadog 应用密钥（必需：True，敏感：True）
- **domain**: Datadog API 域名（必需：False，敏感：False）
- **environment**: 拓扑环境名称（必需：False，敏感：False）
- **oauth_token**: 用于 OAuth 流程（必需：False，敏感：True）

某些范围可能需要才能通过提供商执行特定操作或查询。以下是相关范围及其用例的摘要：
- **events_read**: 读取事件数据。（强制）
- **monitors_read**: 读取监控器（强制）（[文档](https://docs.datadoghq.com/account_management/rbac/permissions/#monitors)）
- **monitors_write**: 写入监控器（[文档](https://docs.datadoghq.com/account_management/rbac/permissions/#monitors)）
- **create_webhooks**: 创建 webhook 集成
- **metrics_read**: 查看自定义指标。
- **logs_read**: 读取日志数据。
- **apm_read**: 读取 APM 数据以创建拓扑。
- **apm_service_catalog_read**: 读取 APM 服务目录以创建拓扑。

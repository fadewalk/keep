## 认证
此提供商需要身份验证。
- **api_server**: openshift api 服务器 url（必需：True，敏感：False）
- **token**: openshift 令牌（必需：True，敏感：True）
- **insecure**: 跳过 TLS 验证（必需：False，敏感：False）

某些范围可能需要才能通过提供商执行特定操作或查询。以下是相关范围及其用例的摘要：
- **connect_to_openshift**: 检查提供的令牌是否可以连接到 openshift 服务器（强制）

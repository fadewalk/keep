**警报管理**使团队能够有效地管理、监控和处理关键警报。

凭借强大且用户友好的界面，Keep 允许用户深入了解其警报、过滤大量数据并快速采取行动以维护系统健康。

<Frame width="100" height="200">
  <img height="10" src="/images/alert_table_1.png" />
</Frame>

警报管理的所有相关内容都可以自定义：

1. **警报表格** - 查看和管理警报。
2. **搜索栏** - 使用 CEL 过滤警报，可保存为"自定义预设"。
3. **筛选器** - 对警报进行切片和筛选。
4. **列和时间** - 为您的预设自定义列和主题。

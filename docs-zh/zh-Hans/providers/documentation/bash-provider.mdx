## 连接 Provider

Bash provider 允许您在工作流程中运行 Bash 命令或脚本。您可以传入任何有效的 Bash 命令，它将在本地环境中执行。

### **云环境限制**
此 Provider 在云环境中被禁用，只能在本地或自托管环境中使用。

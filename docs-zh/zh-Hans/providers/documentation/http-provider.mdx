## 连接 Provider

要连接到 provider，您可以实例化 `HttpProvider` 类的一个实例，提供 `provider_id` 和 `ProviderConfig` 对象。然后您可以调用 `query` 方法来查询 HTTP 端点。

## 注意事项

代码会记录有关正在发送的请求的一些调试信息，包括请求头、请求体和查询参数。此信息不应包含敏感信息，但在生产环境中使用了此 provider 之前，确保这一点很重要。

## 概述

SignalFX Provider 通过与 SignalFX Alerting 的 webhook 无缝集成来增强您的监控和告警功能。这种集成允许您直接从 SignalFX 接收告警，确保您能够及时了解基础设施内的重要事件和指标。

主要功能：
- Webhook 自动检测：自动将 Keep 配置为 SignalFX 内的 Webhook 集成，订阅所有可用的 SignalFX 检测器和规则以进行全面的监控。
- 手动和自动订阅管理：在向新检测器添加 Keep 作为订阅者方面提供灵活性，可以手动完成，也可以通过从 UI 重新运行"设置 webhook"功能来完成维护。

<Tip>如需了解更多信息或帮助，请随时在我们的 Slack 社区上联系。</Tip>

## 连接 Provider
有三种方法连接 SignalFX：
- 推送（手动） - 将 Keep 安装为 Webhook 集成。
- 推送（自动检测） - 让 Keep 将自身检测为 webhook 集成并订阅您的 SignalFx 检测器。
- 拉取 - Keep 将从 SignalFx 拉取告警。


<Tip>安装 SignalFx 的推荐方式是通过推送（自动检测）。使用这种方法，您将受益于推送方法的优势（因为 SignalFx 在 Webhooks 上发送更多上下文）和拉取集成的便利性（只需提供凭据，Keep 会完成其余工作）的结合。</Tip>

在以下部分中，我们将详细说明每种方法。

### 推送（自动检测）
使用这种方法：
1. Keep 将自身安装为 Webhook 集成。
2. Keep 迭代所有检测器和规则，并将自身添加为订阅者

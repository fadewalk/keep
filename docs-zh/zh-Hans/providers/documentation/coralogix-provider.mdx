## 概述

Coralogix 是一个现代化的可观测性平台，提供全面的日志、指标、追踪和安全事件可见性，实现端到端监控。

## 将 Coralogix 连接到 Keep

要将 Coralogix 连接到 Keep，您需要将其配置为 Coralogix 中的 webhook。请按照以下步骤设置集成：

1. 从 Coralogix 工具栏导航到 Data Flow > Outbound Webhooks。

<Frame
    width="100"
    height="200">
    <img height="10" src="/images/coralogix-provider_1.png" />
</Frame>

2. 在 Outbound Webhooks 部分，点击 Generic Webhook。

<Frame
    width="100"
    height="200">
    <img height="10" src="/images/coralogix-provider_2.png" />
</Frame>

3. 点击 Add New。

<Frame
    width="100"
    height="200">
    <img height="10" src="/images/coralogix-provider_3.png" />
</Frame>

4. 输入 webhook 名称并将 URL 设置为 `https://api.keephq.dev/alerts/event/coralogix`。
5. 选择 HTTP 方法（POST）。

<Frame
    width="100"
    height="200">
    <img height="10" src="/images/coralogix-provider_4.png" />
</Frame>

6. 从 [Keep 设置](https://platform.keephq.dev/settings?selectedTab=api-key) 生成具有 webhook 角色的 API 密钥。复制 API 密钥并在下一步中将其粘贴到请求 header 中。

<Frame
    width="100"
    height="200">
    <img height="10" src="/images/coralogix-provider_5.png" />
</Frame>

7. 在 coralogix webhook 配置中添加一个请求 header，键为 "x-api-key"，值为您在 Keep 创建的 API 密钥。

<Frame
    width="100"
    height="200">
    <img height="10" src="/images/coralogix-provider_6.png" />
</Frame>

8. 编辑触发 webhook 时将发送的消息正文（可选）。
9. 保存配置。

<Tip>
  Anthropic Provider 支持查询 Claude 语言模型以进行基于提示的交互。
</Tip>

## 输出

目前，Claude Provider 根据提供的提示输出模型的响应。

## 连接 Provider

要连接 Claude，您需要获取一个 API 密钥：

1. 登录您的 [Anthropic Console](https://console.anthropic.com) 账户。
2. 进入 **API Keys** 部分。
3. 点击 **Create Key** 为 Keep 生成一个新的 API 密钥。

在 Claude Provider 配置的 `authentication` 部分使用生成的 API 密钥。

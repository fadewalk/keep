## 连接 Provider

要连接到 Grok，您需要获取一个 API 密钥：

1. 在 X.AI 平台上订阅 Grok。
2. 在您的 X.AI 账户设置中导航到 API 部分。
3. 为 Keep 生成一个新的 API 密钥。

在 Grok Provider 配置的 `authentication` 部分使用生成的 API 密钥。

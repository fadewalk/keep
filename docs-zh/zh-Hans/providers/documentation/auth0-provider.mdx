## 连接 Provider

Auth0 provider 连接到**身份验证 API** 和**管理 API**，实现诸如基于令牌的身份验证和用户管理等功能。根据您的需求，您可以：
- 使用**身份验证 API** 获取访问令牌、管理用户配置文件或处理多因素身份验证。
- 使用**管理 API** 自动化 Auth0 环境的配置、注册应用程序、管理用户等。

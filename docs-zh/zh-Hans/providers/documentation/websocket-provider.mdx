## 输出
`WebsocketProvider` 的 `query` 函数输出以下格式：

```json
{
  "connection": true,
  "data": "Received data from the websocket"
}
```

`connection` 字段指示 websocket 连接是否成功（`true`）或不成功（`false`）。`data` 字段包含从 websocket 接收到的数据。
如果 `connection` 字段指示连接不成功（`false`），则对象还将包含一个 `error` 字段，其中包含有关失败连接的详细信息。


## 认证参数
Websocket provider 不需要任何特定的认证参数。

## 连接 Provider
要连接 Websocket provider 并执行查询，请按照以下步骤操作：

在您的系统中初始化 provider 和 provider 配置。
使用 WebsocketProvider 的 query 函数与 websocket 进行交互。

有关更多信息，请参阅 [文档](https://websocket-client.readthedocs.io/en/latest/api.html#websocket.WebSocket.send)。

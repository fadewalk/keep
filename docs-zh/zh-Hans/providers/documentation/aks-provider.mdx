## 连接 Provider

要连接到 Azure AKS，请按照以下步骤操作：

1. 登录您的 [Azure](https://azure.microsoft.com/) 账户。
2. 进入您的 Kubernetes 服务页面，点击 `连接` 按钮，然后点击 `打开 Cloud Shell`。
3. 在云 shell 中运行 `az ad sp create-for-rbac --role owner --scopes /subscriptions/<YOUR_SUBSCRIPTION_ID>`，您将收到类似以下的响应：
   ```
    {
      "appId": "xxxxxx-xxxxx-xxxxxx-xxxx",
      "displayName": "azure-cli-2023-11-06-13-00-52",
      "password": "xxxxxxxxxxxxxxxxxxxxxxxxxxxxxx",
      "tenant": "xxxxx-xxxxx-xxxx-xxxxx"
    }
   ```
   在上述 JSON 对象中，`appId` 是 `client_id`，`password` 是 `client_secret`，`tenant` 是 `tenant_id`。

## 注意事项

- 此 Provider 允许您与 Azure AKS 交互以查询 Kubernetes 集群中的资源。

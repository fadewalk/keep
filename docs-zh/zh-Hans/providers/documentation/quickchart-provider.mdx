# QuickChart Provider

## 概述

QuickChart provider 允许基于 Keep 平台内的告警数据生成两种类型的图表：

1. 显示特定指纹告警随时间趋势的折线图。
2. 显示 Keep 收到此指纹告警总数的径向仪表图。

这些图表可用于各种报告、仪表板或告警摘要中，以提供对告警活动和趋势的视觉洞察。

## 连接 Provider

### 不使用 API 密钥使用 QuickChart

QuickChart provider 无需 API 密钥即可生成图表。但是，此用法仅限于基本功能和较低的请求限制。

### 使用 API 密钥使用 QuickChart

要解锁更多高级功能和更高的使用限制，您可以使用 QuickChart API 密钥。以下是获取方法：

1. 访问 [QuickChart](https://quickchart.io/)。
2. 注册免费账户以开始使用。
3. 导航至您的账户设置以找到您的 API 密钥。

获取 API 密钥后，将其添加到 Keep 的 provider 配置中。

## 注意事项

此 provider 旨在在 Keep 内提供灵活的图表生成功能，增强您可视化告警数据和趋势的方式。它非常适合希望快速将告警活动的视觉表示集成到工作流程中的用户。

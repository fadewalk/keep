## 连接 Provider

获取 StatusCake API 密钥

1. 在 [StatusCake](https://www.statuscake.com/) 上创建一个账户。
2. 登录后，进入 [Account Settings](https://app.statuscake.com/User.php) 下的 My Account
3. 在 Manage API Keys 下，生成新的 API 密钥或使用默认密钥。
4. 复制 API 密钥。这将用作 provider 设置中的 `Statuscake API Key`。

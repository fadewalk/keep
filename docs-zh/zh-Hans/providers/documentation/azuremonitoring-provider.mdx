## 概述

Azure Monitor Provider 将 Keep 与 Azure Monitor 集成，允许您在 Keep 平台上接收告警。通过在 Azure 中设置 webhook，您可以确保关键告警被发送到 Keep，从而实现高效的监控和响应。

## 将 Azure Monitor 连接到 Keep

将 Azure Monitor 连接到 Keep 需要在 Azure 中创建一个操作组、添加 webhook 操作，以及配置告警规则以使用新的操作组。

### 步骤 1：导航到操作组
1. 登录您的 Azure 门户。
2. 导航到 **Monitor** > **Alerts** > **Action groups**。

## 概述

Incident.io provider 促进了与 Incident.io API 的交互，允许管理事件。这包括查询特定事件、检索所有事件和管理事件详情的能力。此 provider 将 Keep 的系统与 Incident.io 强大的事件管理平台集成。

## 连接 Provider

### API 密钥

要使用 Incident.io API：
1. 登录您的 Incident.io 账户。
2. 在您的账户设置下导航到 "API Keys" 部分。
3. 生成一个新的 API 密钥或使用现有的密钥。
4. 确保它具有用于读取和管理事件的 `read` 权限。

### 事件端点

Incident.io 事件端点允许查询和管理事件。操作包括检索特定事件详情或获取所有事件的列表。这对于高效监控和响应事件至关重要。

更多详细信息，请参阅 [Incident.io API 文档](https://api-docs.incident.io/)。

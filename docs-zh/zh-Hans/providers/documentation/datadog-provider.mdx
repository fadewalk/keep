## 连接 Provider

### API 密钥

要获取 Datadog API 密钥，请按照以下步骤操作：

1. 登录您的 Datadog 账户。
2. 导航到 "Integrations" 部分。
3. 点击 "API" 选项卡。
4. 生成一个新的 API 密钥。

### App 密钥

要获取 Datadog App 密钥，请按照以下步骤操作：

1. 登录您的 Datadog 账户。
2. 导航到 "Integrations" 部分。
3. 点击 "API" 选项卡。
4. 生成一个新的 App 密钥或使用现有的密钥。

## 指纹

Datadog 中的指纹是根据传入/拉取事件的 `groups` 和 `monitor_id` 字段计算的。

## Webhook 集成修改

Webhook 集成将 Keep 添加为 Datadog 中的监视器。它可以在 "Monitors" 部分找到。
该集成自动获取对 Datadog 中以下范围的访问权限：
- `monitors_read`
- `monitors_write`
- `create_webhooks`

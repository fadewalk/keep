Incident Manager Provider 允许您将 AWS IncidentManager 中的事件推送到 Keep。

<AutoGeneratedSnippet />

## 状态映射

Incident Manager Provider 映射以下状态：

- "OPEN" 映射到 AlertStatus.FIRING
- "RESOLVED" 映射到 AlertStatus.RESOLVED

## 严重性映射

Incident Manager Provider 映射以下严重性：

- 1 映射到 AlertSeverity.CRITICAL
- 2 映射到 AlertSeverity.HIGH
- 3 映射到 AlertSeverity.LOW
- 4 映射到 AlertSeverity.WARNING
- 5 映射到 AlertSeverity.INFO

## 注意事项
1. Incident Manager 仅在响应计划附加了 chatChannel 时才发送通知。在添加 webhook 之前，请确保在响应计划中添加 chatChannel。

## 连接 Provider

在 github 中创建您的个人访问令牌（classic）：
- 在任何页面右上角，点击您的头像，然后点击 **Settings**。
- 在左侧边栏中，点击 **Developer settings**。
- 在左侧边栏中，在 Personal access tokens 下，点击 **Tokens (classic)**。
- 选择 Generate new token，然后点击 Generate new **token (classic)**。
- 在 "Note" 字段中，为您的令牌指定一个描述性名称。
- 要为您的令牌设置过期时间，选择 **Expiration**，然后选择一个默认选项或点击 **Custom** 输入日期。
- 选择您希望授予此令牌的范围。
- 点击 **Generate token**。
- （可选）要复制新令牌到剪贴板，请点击复制按钮。

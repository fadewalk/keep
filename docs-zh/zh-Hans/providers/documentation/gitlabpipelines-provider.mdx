## 连接 Provider

在 GitLab 中创建您的个人访问令牌：
- 在左侧边栏中，选择您的头像。
- 选择 **Edit profile**。
- 在左侧边栏中，选择 **Access Tokens**。
- 选择添加 **new token**。
- 输入令牌的**名称**和**到期日期**。
- 选择所需的范围。
- 选择创建 **personal access token**。

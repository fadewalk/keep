## 连接 Provider

1. 进入 [Asana](https://app.asana.com/0/developer-console)。

<Frame width="100" height="200">
  <img height="10" src="/images/asana-provider_1.png" />
</Frame>

2. 点击 `Create New Personal Access Token`。

<Frame width="100" height="200">
  <img height="10" src="/images/asana-provider_2.png" />
</Frame>

3. 为其命名，然后点击 `Create`。

4. 复制生成的令牌。这将用作 provider 设置中的 `Personal Access Token`。

## 连接 Provider

获取 Ntfy 访问令牌（仅适用于 Ntfy.sh）

1. 在 [Ntfy.sh](https://ntfy.sh/) 上创建账户。
2. 登录后，前往 [Access token](https://ntfy.sh/account) 页面。
3. 点击 `CREATE ACCESS TOKEN`。为其提供标签，选择令牌过期时间，然后点击 `CREATE TOKEN` 按钮。
4. 复制生成的令牌。这将用作 provider 设置中的 `Ntfy Access Token`。

自托管 Ntfy

1. 要自托管 Ntfy，您可以按照此处的说明进行操作 [here](https://docs.ntfy.sh/install/)。
2. 对于自托管 Ntfy，您需要在 provider 设置中提供 `Ntfy Host URL`、`Ntfy Username` 和 `Ntfy Password`，而不是 `Ntfy Access Token`。
3. 为自托管 Ntfy 实例创建新用户，并在 provider 设置中使用生成的用户名和密码。

订阅主题（适用于 Ntfy.sh 和自托管 Ntfy）

1. 登录您的 Ntfy.sh 账户。
2. 点击 `Subscribe to a topic` 按钮，生成主题名称并订阅。
3. 复制生成的主题名称。这将用作 provider 设置中的 `Ntfy Subcription Topic`。
4. 保留主题并配置访问权限（需要 ntfy Pro）

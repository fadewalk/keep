## 概述

ilert provider 促进了与 ilert API 的交互，允许管理事件。这包括创建、更新和解决事件的能力，以及发送自定义事件通知。此 provider 将 Keep 的系统与 ilert 强大的告警和事件管理平台集成。

## 连接 Provider

要将 Keep 与 ilert 集成，请按照以下步骤操作：

1. 登录您的 ilert 账户。
2. 在您的账户设置下导航到 "Alert Sources"。
3. 为 Keep 创建一个新的告警源。
4. 记下此告警源提供的 `ALERT-SOURCE-API-KEY`。

用于 Keep 集成的请求端点是：
(https://api.ilert.com/api/v1/events/keep/{ALERT-SOURCE-API-KEY})

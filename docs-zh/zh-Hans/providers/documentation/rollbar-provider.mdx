## 连接 Provider

1. 在 [Rollbar](https://rollbar.com/) 上创建一个账户。
2. 登录后，导航至您要连接的项目并进入项目设置。
3. 在 Setup 下，转到 Project Access Tokens 并创建具有读取和写入范围的新令牌。
4. 复制生成的令牌。
5. 这将用作 provider 配置中的 `rollbarAccessToken` 参数。

## Webhook 集成修改

您可以通过在 Rollbar 项目中导航至 **Settings > Notifications > Webhook** 来管理 webhook 集成授予的权限。

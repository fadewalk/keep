## 连接 Provider

获取 Splunk API 令牌：
1. 确保您有一个具有必要 [权限](https://docs.splunk.com/Documentation/Splunk/9.2.0/Security/Rolesandcapabilities) 的 Splunk 账户。所需的基本权限是 `list_all_objects` 和 `edit_own_objects`。
2. 获取用于认证 API 请求的 API 令牌。了解如何设置和获取 API 密钥，请阅读更多 [Read More](https://docs.splunk.com/Documentation/Splunk/9.2.0/Security/Setupauthenticationwithtokens)。

识别您的 Splunk 实例详细信息：
1. 确定您希望连接到的 Splunk 实例的主机（IP 地址或主机名）和端口（默认为 Splunk 管理 API 的 8089）。

---
**注意**
请务必按照此 [指南](https://docs.splunk.com/Documentation/Splunk/9.2.0/Alert/ConfigureWebhookAllowList) 配置您的 webhook 允许列表以允许您的 `keep` 部署。

## 连接 Provider

1. 登录您的 Dynatrace 账户并导航到 "Settings" → "Integration" → "Dynatrace API"。
2. 生成具有适当权限（例如读取指标）的 API 令牌。
3. 获取您的环境的 Dynatrace URL。
4. 使用 API 令牌和 Dynatrace URL 配置 Dynatrace provider。

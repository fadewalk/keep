## 连接 Provider

### API 密钥

要获取 Pingdom API 密钥，请按照以下步骤操作：

1. 登录您的 Pingdom 账户。
2. 导航至"Settings"部分。
3. 点击"Pingdom API"选项卡。
4. 生成新的 API 密钥。


## 指纹

Pingdom 中的指纹是根据传入/拉取的事件的 `check_id` 计算的。

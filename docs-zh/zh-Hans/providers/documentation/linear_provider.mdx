## 设置方法

Linear Provider 使用 `api_token` 进行请求授权。您需要提供以下信息：

- **api_token**（必填）：您的 Linear 应用的个人 API 密钥。
  - 获取方式：
    1. 访问 Linear 应用或网站。
    2. 登录您的 Linear 账户。
    3. 导航至账户设置。
    4. 导航至 API 页面。
    5. 在个人 API 密钥部分生成密钥。
    6. 复制生成的 API token。

## 注意事项

- 此 provider 允许您查询给定 Linear 团队的项目。
- 此 provider 允许您为给定项目和团队在 Linear 应用中发送通知（创建 issue）。

---
title: "自动抑制告警"
---

<Info>

工作流链接：[workflow](https://github.com/keephq/keep/blob/main/examples/workflows/autosupress.yml)。

</Info>

此工作流演示如何通过将告警标记为已解除来抑制告警。


说明：
- 触发器：由任何告警激活。
- 动作：通过添加值为 `true` 的 `dismissed` 字段来增强告警。


```yaml
workflow:
  id: autosupress
  description: demonstrates how to automatically suppress alerts
  triggers:
    - type: alert
  actions:
    - name: dismiss-alert
      provider:
        type: mock
        with:
          enrich_alert:
            - key: dismissed
              value: "true"
```

## 第 1 步：登录 AWS 控制台
- 打开您的网络浏览器并导航到 AWS 管理控制台。
- 使用您的 AWS 账户凭证登录。

## 第 2 步：导航到 ECS
- 点击左上角的 "Services" 下拉菜单。
- 从服务列表中选择 "ECS"。

## 第 3 步：创建 3 个任务定义
- 在 ECS 仪表板中，导航到左侧边栏中的 "Task Definitions" 部分。
    <img src="/images/ecs-task-def-create.png" alt="Task Definition" width="200" height="200" />
- 点击 "Create new Task Definition"。
    ![Create new task definition](/images/ecs-task-def-create-new.png)

    ### 任务定义 1（前端 - KeepUI）：

    - 任务定义系列：keep-frontend
        ![Task Definition Family](/images/ecs-task-def-frontend1.png)
    - 按如下方式配置您的容器定义：
        - 基础设施要求：
            - 启动类型：AWS Fargate
            - 操作系统、架构、网络模式：Linux/X86_64
            - 任务大小：
                - CPU：1 vCPU
                - 内存：2 GB
            - 如果您计划使用 secrets manager 等，任务角色和任务执行角色是可选的，那么请创建一个任务执行角色以允许访问您创建的 secret manager。
        ![Infrastructure Requirements](/images/ecs-task-def-frontend2.png)
        - 容器详情：
            - 名称：keep-frontend
            - 镜像 URI：us-central1-docker.pkg.dev/keephq/keep/keep-api:latest
            - 端口映射：
                - 容器端口：3000
                - 协议：TCP
            ![Container Details](/images/ecs-task-def-frontend3.png)
            - 环境变量：（可以是静态的，或者您可以使用 parameter store 或 secrets manager）
                - DATABASE_CONNECTION_STRING
                - AUTH_TYPE
                - KEEP_JWT_SECRET
                - KEEP_DEFAULT_USERNAME
                - KEEP_DEFAULT_PASSWORD
                - SECRET_MANAGER_TYPE
                - SECRET_MANAGER_DIRECTORY
                - USE_NGROK
                - KEEP_API_URL
               （以下变量如果您不想使用 websocket 是可选的）
                - PUSHER_DISABLED
                （以下变量如果您想使用 websocket 是可选的）
                - PUSHER_APP_ID
                - PUSHER_APP_KEY
                - PUSHER_APP_SECRET
                - PUSHER_HOST
                - PUSHER_PORT
            ![Environment Variables](/images/ecs-task-def-frontend4.png)
        - 检查并创建您的任务定义。

                - KEEP_JWT_SECRET

## 第 4 步：创建 Keep 服务
- 在 ECS 仪表板中，导航到左侧边栏中的 "Clusters" 部分。
- 选择您要部署服务的集群。
- 点击 "Create" 按钮旁边的 "Services"。
- 配置您的服务设置。
- 检查并创建您的服务。

## 第 5 步：监控您的服务
- 创建服务后，在 ECS 仪表板中监控其状态。
- 您可以查看任务状态、服务事件和其他指标以确保您的服务正常运行。
